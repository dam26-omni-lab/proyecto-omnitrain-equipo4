# OmniTrain

## Descripción General

OmniTrain es una plataforma corporativa de capacitación diseñada para centralizar procesos de aprendizaje, simulación y seguimiento del desempeño dentro de una organización.

Este proyecto corresponde a la primera versión visual (Landing Page / Home) desarrollada utilizando tecnologías web estándar y preparada para evolucionar posteriormente hacia un sistema más completo con dashboards, simuladores interactivos, indicadores de desempeño (KPIs) y módulos especializados de capacitación.

---

## Objetivo del Proyecto

Desarrollar una interfaz moderna, profesional y responsiva que funcione como punto de entrada para la plataforma OmniTrain.

La aplicación está orientada a empresas que requieren:

- Capacitación continua del personal.
- Seguimiento del progreso de aprendizaje.
- Visualización de indicadores de desempeño.
- Acceso centralizado a distintos módulos de entrenamiento.
- Escalabilidad para integrar simuladores 2D y 3D en futuras versiones.
